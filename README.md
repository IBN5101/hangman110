# (AiC01) Hangman

Hangman is a classic game in which a player thinks of a word and the other player tries to guess that word within a certain amount of attempts.

This is an implementation of the Hangman game, where the computer thinks of a word and the user tries to guess it. 

This project is part of the Ai■■■■ skill bootcamp. This project was completed on 02/12/2023.

## Installation

To execute the script, please install the latest version of Python. As of the time of writing this, the Python version used was 3.12.

## Usage

To play the game, run the final milestone script `milestone_5.py`.
